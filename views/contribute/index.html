Contribute

The data on Data Rights Finder is open. Anyone can contribute to it and anyone can use it and re-use it under the Open Database License.

You can make a contribution of data about an organisation that's not in the database at the moment with this form. Organisations can add data about their own privacy policy. We'll check over all contributions before they're added into the database. All contributions are licensed under the Open Database License.

There's a to-do list of organisations to be added to the database. If there's an organisation you'd like to add to the list send an email to us at {{ settings.contact_email }} to let us know. Or if you have a Github account, you can add the organisation directly to the list.
