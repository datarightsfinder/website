Contribute

We need your help to make information from privacy notices searchable and reusable.

It takes about 15 minutes to complete a submission using this form. Your browser will automatically save your progress.

Don't worry if you can't find all the information we're asking for. When you're finished, you can submit your contribution via GitHub, or email it to us.

To contribute with GitHub:

  1. Sign into GitHub

  2. Create a new file in the Open Data Rights repository

  3. Name the file .json

  4. Copy and paste this code into the editor

  5. Select 'Create a new branch for this commit and start a pull request'

  6. Click 'Propose new file'
